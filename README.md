# NetFlow

## Simple, Buffer based networking library for Roblox.

NetFlow is a minimalist networking library for Roblox. It uses binary buffers to compress your data and supports bidirectional Invoke patterns.

NetFlow는 로블록스를 위한 직관적인 네트워킹 라이브러리입니다. 이진 버퍼를 활용해 성능을 최적화했으며, 단순 이벤트를 넘어 강력한 양방향 인보크(Invoke) 기능을 지원합니다.
