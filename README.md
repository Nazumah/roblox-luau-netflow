# NetFlow

## Simple, Buffer based networking library for Roblox.

NetFlow is a high-performance networking library for Roblox. It leverages advanced binary serialization and deserialization to compress your data, significantly reducing network bandwidth. Beyond simple one-way events, NetFlow provides a robust and reliable bidirectional Invoke system.

NetFlow는 로블록스를 위한 고성능 네트워킹 라이브러리입니다. 효율적인 직렬화(Serialization) 및 역직렬화(Deserialization)를 통해 데이터를 혁신적으로 압축하여 전송하며, 단순한 이벤트 전송을 넘어 강력한 양방향 인보크(Invoke) 기능을 지원합니다.
