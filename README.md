# NetFlow

## Simple, Buffer based networking library for Roblox.

NetFlow is a networking library for Roblox. It uses buffers to compress data and supports powerful **bidirectional Invoke** capabilities beyond simple events.

NetFlow는 로블록스를 위한 네트워킹 라이브러리입니다. 버퍼를 활용해 데이터를 압축하며, 단순 이벤트를 넘어 강력한 **양방향 인보크** 기능을 지원합니다.

## Performance

NetFlow is buffer-based and is significantly faster than non-buffer libraries. This is because it **serializes** outgoing data into the smallest possible size and **deserializes** it on the receiving end. As a result, it optimizes network bandwidth and delivers exceptional speed.

NetFlow는 버퍼 기반으로 비버퍼 라이브러리보다 훨씬 빠릅니다. 이는 데이터를 보낼 때 크기를 최소화하여 **직렬화**하고, 받는 쪽에서 **역직렬화**하기 때문입니다. 결과적으로 네트워크 대역폭을 효율적으로 만들며 매우 빠릅니다.
